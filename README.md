# Trail Climate Tracker

## About
A Python application that integrates climate data APIs to analyze environmental impacts on Pacific Northwest hiking trails. This project combines real-time weather data, historical climate trends, and trail information to help outdoor enthusiasts understand how climate change affects their favorite trails.

## Why This Project?
I'm passionate about both outdoor recreation and sustainable technology. This project allows me to:
- Learn API integration and data processing
- Apply object-oriented design principles
- Work with real-world climate data
- Create something meaningful for the hiking community

## Technologies
- **Language**: Python 3.9+
- **APIs**: NASA POWER, USGS Water Services, OpenWeatherMap
- **Libraries**: requests, matplotlib, numpy, pandas
- **Tools**: Git, pytest
